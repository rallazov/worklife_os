# SPEC - Onboarding Flow
tags: #spec
status: Draft

## Goal
Get a new user to “value” in under 2 minutes.

## Required (Stage A)
1) Timezone
2) Week starts on (Mon/Sun)
3) Primary goal (Job / Family / Work / Business / Mixed)
4) Connect mailbox (Gmail)

## Optional later (Stage B)
- Profession (for suggestions, not gating)
- Modes/presets (Work/Home/Job Defense)
- Digest preferences

## Success criteria
- User reaches Today page with meaningful counts and a clear next action.
